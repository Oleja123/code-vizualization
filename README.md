code-vizualization
Разработка программных модулей в приложении для обучения основам программирования на языках Go, Java, JavaScript (Vue)

## Сервисы

- `semantic-analyzer-service` — семантическая проверка C-кода.
- `interpreter-service` — пошаговое выполнение и snapshot runtime-состояния.
- `cppcheck-analyzer-service` — статический анализ C-кода через cppcheck.
- `flowchart-visualizer` — генерация блок-схем.
- `auth_service` — аутентификация пользователей.
